Steganography Hints

Common techniques and tools for CTF stego challenges. Use these as your checklist.

WORKFLOW: Always start with strings → file → exiftool → then go deeper based on file type.
First Steps (Any File)
Run strings file.png — hidden text often just sitting there
Run file suspicious.jpg — confirm actual file type
Run exiftool image.png — check metadata for hidden data
Run binwalk file.png — extract embedded files
Image Stego (PNG/JPG)
steghide extract -sf image.jpg — try with empty password first
zsteg image.png — LSB analysis for PNGs
stegsolve — visual bit plane analysis
Open in HxD/hex editor — look for appended data after EOF marker
Audio Stego (WAV/MP3)
Open in Audacity — look at spectrogram view (flag often visible!)
View → Spectrogram in Audacity
mp3stego for MP3 files
Check for morse code patterns in the waveform
PDF / Office Files
pdfinfo file.pdf — metadata check
pdf-parser.py — extract objects
Check for white text on white background
Hidden layers in PDFs — Adobe Reader → layers panel
LSB Steganography
Least Significant Bit — data hidden in pixel color LSBs
Tool: zsteg -a image.png (tries all combinations)
Tool: stegsolve — plane 0 of R/G/B channels
Online: stylesuxx.github.io/steganography
File Signatures (Magic Bytes)
PNG: 89 50 4E 47
JPG: FF D8 FF
ZIP: 50 4B 03 04
PDF: 25 50 44 46
GIF: 47 49 46 38
If extension is wrong, rename using true magic bytes
Quick Reference Cheatsheet

Pattern recognition for CTF crypto. See a string → instantly know what it is.

BASE64
Base64
Chars: A-Z a-z 0-9 + / =
Ends with = or == padding
Length always multiple of 4
Example: SGVsbG8gV29ybGQ=
BASE32
Base32
Chars: A-Z 2-7 = (uppercase only)
Always ends with = padding
Example: JBSWY3DPEBLW64TMMQ======
HEX
Hexadecimal
Chars: 0-9 a-f only
Often prefixed with 0x
Length always even
Example: 48656c6c6f = Hello
HASH
Hash Lengths
MD5: 32 hex chars
SHA1: 40 hex chars
SHA256: 64 hex chars
SHA512: 128 hex chars
bcrypt: starts with $2b$
CLASSICAL
Classical Cipher Hints
Caesar: letter frequency still visible, just shifted
ROT13: Caesar with shift 13 (letters only)
Atbash: A→Z B→Y pattern
Vigenère: repeating key, harder to spot
Morse: only . - / characters
OTHER
Other Encodings
URL encode: %xx format, %20 = space
Binary: only 0 and 1, groups of 8
Octal: digits 0-7 only
ASCII codes: numbers 32–126
Brainfuck: only +-[]<>., chars
